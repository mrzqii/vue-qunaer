#### 这个项目是在工作之余写的，参考了网上的教程。里面涉及的内容还是比较丰富，子组件父组件之间的通讯，数据传递，在各个生命周期绑定事件都有所涉及。在看过vue的文档以后这个是比较合适的练手项目。

#### 技术栈：
vue2 + vue-router + webpack + ES6 + vue-axios + styl + flex

#### 项目运行
```
git clone https://github.com/mrzqii/vue-qunaer.git

cd vue-qunaer

npm install

npm run dev
```
#### 项目文件里面有详细的注释便于理解。
